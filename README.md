# 🎓 Student Management System (Java)

A console-based Student Management System built in Java using Object-Oriented Programming (OOP), CRUD operations, and ArrayList.

## 🚀 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- ❌ Delete Student Record
- 📜 Menu-driven Console Interface

---

## 🛠️ Tech Stack

- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
- CRUD Operations
